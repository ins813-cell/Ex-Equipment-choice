# Ex-Подбор (Android)

MVP Android-приложение для предварительного подбора и проверки взрывозащищённого оборудования (РФ/ЕАЭС).

## Что реализовано
- Offline-first каталог (Room).
- Мастер подбора по условиям среды/зоны/подгруппы/T-классу.
- Парсер Ex-маркировки с `PARSED_OK/PARSED_PARTIAL/NOT_PARSED`.
- Rule Engine с принципом безопасной неопределённости (`NEED_MORE_DATA` при неполном парсинге).
- Проверка реестров через backend API и локальный кэш со статусом `stale`.
- Генерация PDF отчёта c дисклеймером и trace правил.
- Unit тесты парсера и rule engine.

## Запуск
1. Открыть проект в Android Studio Iguana+.
2. Sync Gradle.
3. Запустить `app`.

## Тесты
```bash
./gradlew test
```
