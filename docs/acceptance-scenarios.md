# Приёмочные сценарии (минимум 15)

1. GAS зона 1 + IIB + T4 -> список и чек-лист.
2. GAS зона 0 + IIC + T4 -> фильтрация по совместимости.
3. DUST зона 21 + IIIB -> список подходящих позиций.
4. Нет совпадений -> пустой список + сообщение, какие поля ограничили выдачу.
5. Поиск по модели -> результат < 1 сек на индексе.
6. Поиск по производителю -> результат < 1 сек.
7. Ручной ввод без T-класса -> NEED_MORE_DATA.
8. NOT_PARSED -> предложение ручного ввода полей.
9. PARSED_PARTIAL -> запрет OK.
10. OCR распознал строку -> пользователь редактирует, правка в CheckLog.
11. Проверка сертификата онлайн -> checked_at сохранён.
12. Реестр недоступен -> показан кэш + "не актуализировано".
13. Создание проекта офлайн -> успешно.
14. Генерация PDF <= 10 сек с версиями RuleSet/каталога.
15. Экспорт журнала в CSV.
